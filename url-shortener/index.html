What is URL Shortener?
URL Shortener is a free service for shortening web addresses and other URLs. This is useful in many circumstances e.g. when sending an address to a phone via SMS and character count is limited.

As an example you can turn:

http://maps.google.co.uk/maps?f=q&source=s_q&hl=en&geocode=&q=louth&sll=53.800651,-4.064941&sspn=33.219383,38.803711&ie=UTF8&hq=&hnear=Louth,+United+Kingdom&z=14
into something much shorter like:

http://url-shortner.appspot.com/aw34

You can also pick a specific meaningful shortened address if you want to, such as:

http://url-shortner.appspot.com/myfb

How to use URL Shortener?

Type or paste the address you want to shorten into the box on our homepage and push the "Shorten" button. It's that simple.
If you want to pick a shortened URL instead of having us generate one for you, before pressing "Shorten", click on "Specify your desired URL" and type your choice in the additional box that appears.

Uses for URL shortener

Here are a few of the most common uses for URL Shortener: -

1) Lower the character count when texting web address to a mobile phone via SMS.

2) Lower the character count for instant messaging services or social networking sites like Twitter where space is limited.

3) Work around problems with certain forum or blog software and some older email clients which may not handle very long links or links containing non-English characters correctly or may hyperlink the wrong parts of complex links.

4) Make URLs easier to read aloud using our custom shortened URL facility or our pronounceable URL facility.
